    温度和气体动理论

    • 热学研究的对象是热力学系统，宏观上看与温度有关，微观上看与粒子的无规则运动(热运动)有关。

    平衡态¶

    • 在不受外界影响的条件下（与外界无任何形式的物质与能量交换），系统的宏观性质不随时间变化的状态（动平衡）

    • 宏观量

      • 表征系统宏观性质的物理量（可直接测量）。
    • 微观量
      • 描写单个微观粒子运动状态的物理量（只能间接测量）。

    • 宏观量总是相应微观量的统计平均值

    • 态参量：

      • 描写平衡态的宏观物理量，一组态参量可以共同描述一个平衡态(温度，压强等)
    • 物态方程：
      • 态参量之间的函数

    温度¶

    • 热平衡态：两个系统热接触下，相当长时间后达到的共同平衡态。

    热力学第零定律¶

    • 分别与第三个系统处于同一热平衡态的两个系统必然也处于热平衡。
    • 一切处于同一热平衡态的系统有相同的温度！
    • 处于同一热平衡态下的热力学系统所具有的共同的宏观性质。
    • 温度取决于系统内部的热运动（对质心的运动）状态

    温标¶

    • 温标：温度的数值标度

    摄氏温标¶

    • 摄氏温标\(t\)以1atm下的水的冰点和沸点作为定点，规定为0，100度。

    理想气体温标¶

    • 理想气体：在各种压强下都严格遵守玻意耳定律的气体！
    • 玻意耳定律：一定质量的气体，在一定温度下，其压强和体积的乘积是个常量,\(pV = Const\)(温度不变)
    • 标准温度定点：水的三相点(triple point) ，水、冰和水汽共存达到平衡态的温度，此时水汽的压强为609Pa。\(T_3 = 273.16K\)
    \[T = T_3\frac{pV}{p_3V_3}=273.16\frac{pV}{p_3V_3}\]

    热力学温标¶

    • 不依赖测温物质及其测温属性的温标。
    • 与摄氏温度 \(t\) 的转换关系
    \[T = t+273.15\]

    理想气体状态方程¶

    • 对于一定质量（\(\nu\space mol\)）的理想气体,有以下方程成立
    • 普适气体常数\(R = 8.31 (J/(mol \cdot K))\)
    \[pV = \nu RT\]
    • 若气体质量为\(m\)，气体摩尔质量为\(M\)，有
    \[pV = \frac{m}{M}RT\]
    • 若系统内的分子数为\(N\)，有
    • 玻尔兹曼常量\(k = \frac{R}{N_A} = 1.38\times 10^{-23} J/K\)
    \[pV = NkT\]
    • 引入\(n = \frac{N}{V}\)，\(n\)是气体分子数密度，表示单位体积内分子个数。
    \[p = \frac{N}{V}kT = nkT\]

    气体分子的无规则运动¶

    • 平均碰撞频率 \(\overline z\)：单位时间内一个气体分子与其它分子碰撞的平均次数
    • 平均自由程 \(\overline \lambda\)：气体分子在相邻两次碰撞间飞行的平均路程\(\(\overline \lambda = \frac{\overline v \Delta t}{\overline z \Delta t} = \frac{\overline v }{\overline z}\)\)

    平均碰撞频率与平均速率的关系¶

    理想气体，在平衡态下,假定： - 只有一种分子； - 分子可视作直径为 \(d\) 的刚球 - 被考虑的分子以平均相对速率 \(\overline u\) 运动，其余的分子静止 - 显然，只有中心在碰撞截面沿\(\vec u\) 扫过的柱体内的分子能碰撞 - 碰撞截面\(\sigma = \pi d^2\)，\(d\)为分子直径

    \[\overline z = \frac{n\overline u\sigma\Delta t}{\Delta t} = n\overline u\sigma\]
    • 碰撞的两个分子的速度方向随机在0~180度变化，故\(\overline \theta = \frac{\pi}{2}\)，且分子的运动速率都相等，\(\overline u = \sqrt 2 \overline v\)
    \[\overline z = \sqrt2n \pi d^2 \overline v\]

    平均自由程与压强、温度的关系¶

    • 遵循以上假设，可以得到：
    \[\overline \lambda = \frac{1}{\sqrt2 n\pi d^2}=\frac{kT}{\sqrt2 p\pi d^2}\]

    理想气体的压强¶

    • 气体动理论的基本观点：
      • 1.宏观物体由大量分子、原子构成，有一定的间隙；
      • 2.分子永不停息地作无规则运动 — 热运动
      • 3.分子间有一定相互作用力
    • 理想气体的微观假设：
      • 大小：分子线度<<分子间平均距离，可以忽略不计
      • 分子力：除碰撞的瞬间，在分子之间、分子与器壁之间无作用力
      • 碰撞性质 ：弹性碰撞
      • 服从经典力学规律
    • 关于大量分子的统计假设（对平衡态）
      • 无外场时，分子在各处出现的概率相同。\(n = \frac{dN}{dV} =\frac{N}{V}=C\)
      • 由于碰撞，分子可以有各种不同的速度，速度取向各方向等概率.分子速度按方向的分布是均匀的。
    \[\overline {v_x^2}=\overline {v_y^2}=\overline {v_z^2} = \frac{1}{3}\overline {v^2}\]
    • 以上的假设都是统计性假设，只适用于大量分子的集体。以及上边涉及的平均值是统计平均值，只对大量分子的集体有意义。
    • 统计规律有涨落 （fluctuation），统计对象的数量越大，涨落越小.

    压强公式推导¶

    • 前提：平衡态，忽略重力，分子看成质点（只考虑分子的平动）
    • 把所有分子按速度区间分成若干组，第\(i\)组分子的速度在\([\vec v_i,\vec v_i+d\vec v_i]\)内,可以认为都基本上是\(\vec v_i\)，以\(n_i\)表示这一组分子的数密度，\(n = \sum n_i\)
    • 取容器壁(垂直于x轴)上一个小面元\(dA\)，一个分子对其冲量为\(2m\vec v_{ix}\)
    • \(dt\)时间内，有\(n_iv_{ix}dAdt\)个速度为\(v_i\)的分子与其相撞，故总冲量\(n_iv_{ix}dAdt(2mv_{ix})\)
    • 对所有\(v_x>0\)的分子求和，故可知\(dI_i = \sum_{v_x >0} 2mn_iv^2_{ix}dAdt\)
    • 由于分子运动的随机性，\(dI = \frac{1}{2}\sum dI_i = \sum_i mn_iv^2_{ix}dAdt\)
    • 由于\(p = \frac{dF}{dA} = \frac{dI}{dtdA}\),
    \[p = \sum_i mn_iv^2_{ix} =\sum_i m\frac{N_i}{V}v^2_{ix} = \frac{mN}{V} \sum \frac{N_i}{N}v_{ix}^2 \]
    \[p= \frac{mN}{V} \overline{v_{ix}^2} =nm\overline{v_{ix}^2} =\frac{1}{3}nm\overline{v^2}\]
    • 定义分子平均平动动能\(\overline {\epsilon_t} = \frac{1}{2}m \overline{v^2}\)，则
    \[p=\frac{2}{3}n\overline{\epsilon_t}\]

    温度的微观意义¶

    由

    \[p=\frac{2}{3}n\overline{\epsilon_t} = nkT\]
    \[\overline{\epsilon_t} = \frac{3}{2}kT\]
    • 说明\(T\)是大量分子热运动平均平动动能的量度

    由于

    \[\frac{1}{2} m \overline{v^2} = \frac{3}{2}kT\]

    可以得到

    \[\overline{v^2} = \frac{3kT}{m}\]
    • 方均根速率：
    \[\sqrt{\overline{v^2}} = \sqrt{\frac{3kT}{m}} = \sqrt{\frac{3RT}{M}}\]

    能量均分定理¶

    • 自由度：决定物体空间位置的独立坐标数，用 \(i\) 表示
    平动自由度 转动自由度 振动自由度 总自由度
    单原子分子 3 0 0 3
    双原子分子 3 2 1 6
    多原子分子 3 3 3N-6 3N
    刚性双原子分子 3 2 0 5
    刚性多原子分子 3 3 0 6
    • 能量均分定理：在温度为\(T\)的平衡态下，分子热运动的每一个自由度所对应的平均动能都等于\(\frac{1}{2}kT\)。
    • 若一个气体分子的总自由度为\(i\)，则平均总动能为\(\overline{\epsilon_k} =\frac{i}{2}kT\).
    • 若理想气体分子总数\(N\),则内能为\(E =N\overline{\epsilon_k} =N\frac{i}{2}kT=\frac{i}{2}\nu RT\)
    • 能量均分定理不仅适用于气体，也适用于液体和固体，甚至适用于任何具有统计规律的系统

    麦克斯韦分布律¶

    • 速度分布函数：
      • 物理意义：速率在速率v所在的单位速率区间的分子数占总数的百分比
      • 对于一个分子来说，f (v) 就是分子处于速率v 附近单位速率区间的概率.
      • 设\(dN_v\)为速率\(v+dv\)中的分子数，\(N\)为总分子数。
      • 速率分布函数\(f(v) = \frac{dN_v}{Ndv}\)
    • 所有分布函数必须满足归一化条件：
    \[\int_0^N \frac{dN_v}{N} = \int_0^{\infty} f(v)dv = 1\]
    • 麦克斯韦速率分布函数
    \[f(v) = 4\pi(\frac{m}{2\pi kT})^{\frac{3}{2}}v^2e^{-\frac{mv^2}{2kT}}\]
    • 最概然速率：相应于速率分布函数\(f(v)\)的极大值的速率\(v_p\)称为最概然速率.
    \[v_p = \sqrt{\frac{2kT}{m}}= \sqrt{\frac{2RT}{M}}\]
    \[f(v_p) = \frac{1}{e}\sqrt{\frac{8m}{\pi kT}}\]
    • 温度越高, 速率大的分子数比例越大, 气体分子的热运动越激烈.
    • 平均速率：
    \[\overline v = \int_0^{\infty}vf(v)dv = \sqrt{\frac{8kT}{\pi m}}=\sqrt{\frac{8RT}{\pi M}}\]
    \[\overline{v^2} = \int_0^{\infty}v^2f(v)dv = \frac{3kT}{m}\]
    \[\sqrt{\overline{v^2}} =\sqrt{\frac{3kT}{m}}\]

    不同平均速率的使用情况¶

    • 讨论分子平均平动动能时用方均根速率
    • 讨论分子碰撞问题时用平均速率
    • 讨论分子的速率分布时用最概然速率

    实际气体等温线¶

    • 若液体与其蒸汽共存而且能处于平衡的状态，这时的蒸汽为饱和蒸气。对应的压强为饱和蒸气压。
    • 若在一条等温线下，没有液气共存的转变过程，而是缩为一点\(K\)，则该温度为临界温度，\(K\)为临界点。
    • 在一定温度下，饱和蒸汽压是定值，与体积无关。
